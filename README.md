raytracing
==========
Raytracer - implemented in C++.
![Screenshot-1](img/screenshot-1.png?raw=true)
![Screenshot-2](img/screenshot-2.png?raw=true)


## Dependencies

 * OpenGL and GLUT.
 * Eigen.
 * C++ compiler.

## Build and Usage
```
cd path/to/raytracer
make
./raytracer image-XX
(image is in img/ folder)
```
